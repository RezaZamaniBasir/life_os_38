# Life OS - Age 38 PWA v3 Daily

تغییرات:
- اضافه شدن لغت انگلیسی روزانه
- اضافه شدن پیام انگیزشی و روانشناسی روزانه
- حفظ قابلیت چند تجربه فرهنگی در یک روز
- حفظ قابلیت چند موضوع یادگیری در یک روز
- حذف تب اپ از پایین صفحه

آپدیت در GitHub:
فایل‌های داخل این پوشه را در ریشه repository قبلی جایگزین کن:
- index.html
- manifest.webmanifest
- service-worker.js
- README.md
- icons/

اگر روی گوشی نسخه قدیمی را دیدی:
- یک بار صفحه را Refresh کن.
- اگر نشد، در Chrome داده‌های سایت را Clear کن.
- چون PWA cache دارد، گاهی باید چند دقیقه صبر کنی.
