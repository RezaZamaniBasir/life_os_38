# Life OS 38 - Install Fixed PWA

فایل‌های داخل این پوشه را در ریشه repository آپلود کن:
- index.html
- manifest.webmanifest
- service-worker.js
- icons/

داده‌ها داخل GitHub ذخیره نمی‌شوند. داده‌ها در localStorage مرورگر/اپ روی همان گوشی ذخیره می‌شوند، مگر اینکه خودت فایل JSON/CSV بکاپ را داخل repository آپلود کنی.
